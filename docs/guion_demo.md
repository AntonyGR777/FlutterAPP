# Guion para video demo movil

1. Abrir la app e iniciar sesion.
2. Mostrar la pantalla de Inicio con controles:
   slider, switch, radio buttons, picker, iconos y botones de navegacion.
3. Entrar a Gestionar meks:
   cargar la lista desde la API, agregar un mek, editarlo, ver informacion y eliminarlo.
4. Entrar a Archivo Local:
   mostrar lectura de `assets/datos.txt`, presionar escribir archivo local y leer el archivo creado.
5. En Gestionar meks:
   mostrar que se usan GET, POST, PUT y DELETE desde la API.
6. Mostrar que al crear/actualizar/eliminar un mek aparece una notificacion del sistema.
7. Entrar a Tareas Asincronicas:
   mostrar FutureBuilder, ejecutar Async/Await, Compute/Isolate y Timer.periodic.
8. Entrar a SQLite usuarios:
   mostrar la lectura de usuarios guardados.
9. Cerrar mostrando que la app mantiene tema gris y navegacion entre pantallas.
